    My Recent Projects

    User Registration System

    with Serverless Email Notifications and Automated Infrastructure Deployment

    This project involved building and deploying a scalable web app using modern cloud and DevOps practices. Key features included:

  • Serverless Email Notifications: Leveraged Google Pub/Sub to trigger a serverless cloud function that sends email notifications from a custom domain
  • Custom VM Image Creation: Using Packer and GitHub Actions, automated the creation of a custom VM image. The image included shell scripts to install required dependencies, as well as a custom systemd service file for app startup
  • Infrastructure as Code: managed the deployment of the app infra on Google Cloud Platform (GCP) using Terraform with a load balancer in front to distribute traffic evenly across instances
  • Performance Testing: conducted extensive performance testing using Postman to simulate heavy load scenarios

    MyDear App

    MyDear is a full-stack application that lets parents capture and preserve their children's precious moments and milestones.

  • Parents start working on a digital archive by creating albums which can be viewed in different modes
  • Each photo carries a memory and a story or note can be added to it as well as tags for later search
  • Achievements in childhood are reminders of the child's growth and can be added to the archive
  • Heartfelt letters and diary entries to their children, waiting to be discovered when they come of age.
